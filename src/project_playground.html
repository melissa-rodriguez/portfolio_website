Playground

A SKETCHBOOK REPOSITORY

The 'playground' repo is a living archive of creative coding experiments. A custom script automatically updates the README, creating an interactive gallery of all my sketches with each image linked directly to its corresponding sketch folder. It started as a way to track my daily Processing sketches without having to remember what I named each one. You know how it is - make something cool, forget what you called it, never find it again. Not anymore!

It has since evolved into a testing ground for new ideas. Here, I prototype concepts before implementing them in larger projects.

Developer / Artist

2018 - Present

My creative coding journey has evolved significantly over time. In my early days with Processing, I primarily focused on crafting looped animations, which served as an excellent foundation for learning coding fundamentals, often through exploring trigonometric functions and circular motions. As I transitioned to p5.js, my approach shifted towards creating static images. These now serve as building blocks for more complex, long-form generative artworks. This evolution reflects not only my growing technical skills but also a deeper exploration of generative art principles.
